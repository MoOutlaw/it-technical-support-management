# IT Technical Support & Management Project

A professional project report covering the full planning, execution, and delivery of an IT technical support management system. The project demonstrates practical understanding of service desk operations, incident management workflows, and structured project delivery using industry-aligned frameworks.

Completed as part of a Technical Support and IT Management module at Aston University.

---

## Module Context

**Module:** Technical Support and IT Management
**Institution:** Aston University
**Year:** 1st Year, BSc Computer Science

---

## Project Overview

This project covers the complete lifecycle of an IT support management system, from initial planning through to documented delivery. The work demonstrates knowledge of how IT support functions are structured and managed in professional environments, with reference to ITIL-aligned practices.

**Areas covered:**

- Service desk processes and escalation procedures — defining how support requests are logged, categorised, and escalated based on priority and complexity
- Incident and problem management workflows — distinguishing between reactive incident resolution and proactive problem management to reduce recurring issues
- Project timeline and resource planning — breaking down deliverables into manageable phases with assigned responsibilities and realistic timeframes
- Gantt chart scheduling — visual tracking of tasks, dependencies, and milestones across the project lifecycle
